About the project
-----------------

A small vendingmachine written in bash mainly using functions.

Project goal
------------

The goal of this project was to create my first ever game with the WinForms project.

Features
- Control a snake with wasd or a game controller.
- Eat apples to grow.
- If you lose you can immediatly restart or quit the game.

What I Practiced
----------------

- Using a timer for ticks as an ingame clock
- RawGameController input to enable gaming with a controller.

How to download and execute
---------------

1. Go to the top right of the project and download the code in a ZIP file
2. Unzip the ZIP file
3. Check that Visual Studio 2026 is installed.
4. open the .xslnx in the folder.
5. press F5 to run the code.
